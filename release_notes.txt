## What's New

- **--fix flag**: Automatically fixes common issues in .env files. Sorts keys alphabetically, removes duplicate keys (keeps last value), and creates a backup of the original file before writing changes.
- **--no-backup flag**: Skips backup creation when using --fix.
- **New fixer module** with `fix_env_file()` function for programmatic use.
