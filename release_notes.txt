## What's New

- **.envguard.toml config file**: Define default settings for strict, check_secrets, format, no_color, no_backup, envignore, and example in a TOML file. CLI flags override config file values.
- **--config PATH flag**: Specify a custom config file location.
- **New config module** with `load_config()` and `merge_config()` functions for programmatic use.
- **pre-commit hook support**: Use envguard as a pre-commit hook via `.pre-commit-hooks.yaml`.
