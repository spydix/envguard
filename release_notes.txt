Bugfix release.

Fixed:
- Inline comments in unquoted values are now properly stripped. Previously a # anywhere in the value would truncate it, even mid-word. Now only a # preceded by whitespace is treated as a comment marker for unquoted values. Quoted values preserve # as part of the value.
- URLs with hash fragments (http://example.com/page#section) are no longer truncated.

Added:
- Tests for inline comment stripping (unquoted, quoted, URLs, hash-without-space).

Tests: 39 passing.
