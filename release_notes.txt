Feature release.

Added:
- Secret detection module with patterns for:
  - AWS Access Keys (AKIA...)
  - GitHub tokens (ghp_..., ghs_...)
  - GitLab tokens (glpat-...)
  - Slack tokens (xox...)
  - JWT tokens
  - Private key blocks (-----BEGIN...PRIVATE KEY-----)
  - Generic API keys, passwords, secrets, tokens
- --check-secrets CLI flag to enable secret scanning
- Severity levels: high and medium
- 8 tests for secret detection patterns

Known issues:
- Password pattern uses case-insensitive regex which matches the word "password" even in comments on the same line. False positives possible. Fix coming in 0.3.1.
- Values containing # (inline comments) are truncated at the # symbol. Fix coming in 0.3.2.

Tests: 29 passing.
